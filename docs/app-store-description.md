## 説明文

### 日本語

```
声で残す、ゆるい日記習慣。

24時間で消える、30秒の音声日記SNS。
寝る前のほんの30秒で、今日を声に残そう。

► 昨夜のできごと

ベッドに入って、ふと思い出した。
「今日、電車で席譲ってもらったな」

書くほどじゃない。
でも、なんか残しておきたい。

録音ボタンを押して、30秒。
「今日さ、電車で席譲ってもらって...」

それだけ。
タイピングなし、編集なし。
ただ、話すだけ。

翌朝には消えてる。
だから、気負わない。

► 日記が続かなかった、あなたへ

「毎日書く」って決めたノート、
3日で白紙になってませんか?

声なら、違う。

布団の中で目を閉じたまま、
今日あったことをぼそっと話すだけ。

30秒で終わる。
だから、続く。

► 消えるから、本音になれる

ずっと残ると思うと構えてしまう。
でも、24時間で消えるなら?

「今日、上司にイラっとした」
「好きな人と目が合った気がする」
「なんか知らんけど幸せだった」

完璧じゃなくていい。
まとまってなくていい。
今日の気持ちを、そのまま声に。

► 文字じゃ伝わらない「今日」がある

「疲れた」って文字で打つのと、
「つかれた...」って声で言うのは、全然違う。

声には温度がある。
テンションが乗る。
ため息も、笑い声も、そのまま届く。

あなたの声だから、伝わるものがある。

► 誰かの夜を覗く、不思議な心地よさ

フォローしている人の声を聴く。

「今日、猫カフェ行ってきた」
「明日のプレゼン緊張する...」
「やっと週末だ〜」

知らない誰かの、何気ない1日。
それを聴きながら眠りにつく。

返信しなくていい。
いいねしなくてもいい。
ただ、聴くだけ。

この距離感が、ちょうどいい。

► SNS疲れしてる人へ

映えなくていい。
バズらなくていい。
数字を気にしなくていい。

ここには「いいね」の数もない。
あるのは、声だけ。

誰かに聴いてもらえたら、それでいい。
そんなゆるさが、ここにはある。

► 仕様

■ 録音時間: 最大30秒
■ 投稿の有効期限: 24時間
■ 価格: 基本無料

► 最後に

特別なことを話さなくていい。
面白いことを言わなくていい。

今日あった小さなこと。
それを声にするだけ。

Voicelet
声で残す、ゆるい日記習慣。
```
