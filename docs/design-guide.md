## 1. 美的方向性（Aesthetic Direction）

### コアテーマ: 「夜のささやき」（Nocturnal Whisper）

寝る前の静かな時間に声を残すアプリとして、**夜の穏やかさと儚い光**を表現する。

| 要素 | 方向性 |
|------|--------|
| **トーン** | Soft / Dreamy / Nocturnal / Organic |
| **雰囲気** | 夜のベランダで星を見ながらひとりごとを言う瞬間 |
| **キーワード** | 消えゆく美しさ、柔らかな光、自然な声、夜の静寂 |
| **避けるべき印象** | 派手、賑やか、テック感、冷たい、無機質 |

### デザイン原則

```
1. 【消えゆく儚さ】24時間で消える投稿 → フェード、ボケ、透明感で表現
2. 【声の温かみ】文字より自然な声 → 丸み、有機的な形状
3. 【夜の習慣】寝る前の使用 → 目に優しいダークテーマ、低コントラスト
4. 【気軽さ】完璧じゃなくていい → 余白を多めに、圧迫感を与えない
```

### 記憶に残るシグネチャ要素

- **光のパーティクル**: 背景に浮遊するボケライト（アイコンのモチーフを継承）
- **葉のアクセント**: 自然さ・成長を表す緑の葉モチーフ
- **波形ビジュアライザー**: 声を視覚化する有機的な波形

## 2. カラーシステム

### 基本的な考え方

- **ベース**: 深い紫の夜空（純粋な黒は使わない）
- **アクセント**: 葉の緑（アイコンから継承）
- **光**: 暖色〜寒色のボケたパーティクル

### カラーパレット

```css
:root {
  /* === ベースカラー（夜空） === */
  --color-bg-primary: #1a1625;      /* 深い紫の夜空 */
  --color-bg-secondary: #241f31;    /* やや明るい紫 */
  --color-bg-tertiary: #2d2640;     /* カード背景 */
  --color-bg-elevated: #362f4a;     /* 浮き上がった要素 */
  
  /* === アクセントカラー（葉・光） === */
  --color-accent-primary: #7ecfb3;   /* メインの緑（葉） */
  --color-accent-secondary: #9ee7d1; /* 明るい緑（ホバー） */
  --color-accent-tertiary: #5fb89a;  /* 暗い緑（押下） */
  
  /* === ライトパーティクル === */
  --color-particle-warm: #f5d78e;    /* 暖かい光 */
  --color-particle-cool: #a8d4f0;    /* 涼しい光 */
  --color-particle-pink: #e8b4d6;    /* 柔らかいピンク */
  
  /* === テキストカラー === */
  --color-text-primary: #f0eef5;     /* メインテキスト */
  --color-text-secondary: #b8b3c7;   /* サブテキスト */
  --color-text-tertiary: #7d7891;    /* プレースホルダー */
  --color-text-inverse: #1a1625;     /* 反転テキスト */
  
  /* === セマンティックカラー === */
  --color-success: #7ecfb3;
  --color-warning: #f5d78e;
  --color-error: #e88b8b;
  --color-info: #a8d4f0;
  
  /* === 状態カラー === */
  --color-listened: #4a4459;         /* 視聴済み（グレーアウト） */
  --color-unlistened: #7ecfb3;       /* 未視聴（カラーリング） */
}
```

### グラデーション

```css

  /* 背景グラデーション */
  --gradient-bg-main: linear-gradient(180deg, #1a1625 0%, #241f31 50%, #1a1625 100%);
  
  /* アクセントグラデーション */
  --gradient-accent: linear-gradient(135deg, #7ecfb3 0%, #5fb89a 100%);
  
  /* 録音中（赤系） */
  --gradient-recording: linear-gradient(135deg, #e88b8b 0%, #d46a6a 100%);
}
```

### シャドウ & グロー

```css

  /* ソフトシャドウ */

  --shadow-md: 0 4px 16px rgba(0, 0, 0, 0.2);

  --glow-recording: 0 0 30px rgba(232, 139, 139, 0.4);
}
```

## 3. タイポグラフィ

### フォント選定方針

- **日本語**: 丸ゴシック系で柔らかさと読みやすさを両立
- **英数字**: 丸みのある幾何学的サンセリフで夜の穏やかさを表現
- **数字（タイマー等）**: 等幅フォントで視認性を確保

### フォントファミリー

```css
:root {
  /* 日本語メインフォント */
  --font-primary: 'Zen Maru Gothic', 'Hiragino Maru Gothic ProN', sans-serif;
  
  /* 英数字ディスプレイフォント */
  --font-display: 'Quicksand', 'Zen Maru Gothic', sans-serif;
  
  /* モノスペース（タイマー等） */
  --font-mono: 'JetBrains Mono', 'SF Mono', monospace;
}
```

### タイプスケール

```css
:root {
```

## 5. モーション & アニメーション

### 基本方針

- **穏やか**: 急な動きを避け、ゆったりとしたイージングを使用
- **儚さ**: フェードイン/アウトを多用し、「消えゆく」感覚を演出
- **有機的**: 波形やパーティクルは自然な揺らぎを持たせる

### トランジション

```css
:root {
  --transition-fast: 150ms ease;
  --transition-normal: 250ms ease;
  --transition-slow: 400ms ease;
  --transition-bounce: 400ms cubic-bezier(0.34, 1.56, 0.64, 1);
}
```

### 主要なモーションパターン

| パターン | 用途 | 特徴 |
|---------|------|------|
| **フェードイン** | 画面遷移、要素表示 | opacity 0→1、ゆっくり |
| **スライドアップ** | モーダル、トースト | 下から上へ、フェードと併用 |
| **パルス** | 録音中ボタン | 拡大縮小の繰り返し、赤いグロー |
| **浮遊** | 背景パーティクル | ゆっくり上下、回転も加える |
| **波形** | 音声ビジュアライザー | 縦方向のスケール変化 |

## 6. ビジュアル表現

### 背景の演出

- **グラデーション**: 単色ではなく、微妙なグラデーションで奥行きを出す
- **パーティクル**: アイコンのボケライトを継承し、背景に浮遊する光の粒を配置
- **透明度**: 要素の重なりで深度を表現

### アイコンスタイル

- **スタイル**: Rounded / Soft（角丸、線幅2px）
- **推奨セット**: Lucide Icons または Phosphor Icons（Rounded）
- **カラー**: 通常は `--color-text-secondary`、アクティブ時は `--color-accent-primary`

### 状態の表現

| 状態 | 表現方法 |
|------|---------|
| **未視聴** | アクセントカラーのリング/ハイライト |
| **視聴済み** | グレーアウト（opacity: 0.5〜0.6） |
| **録音中** | 赤系グラデーション + パルスアニメーション + グロー |
| **ローディング** | 3つのドットが順番にフェード |
| **無効** | opacity: 0.4 |

## 7. アクセシビリティ

### コントラスト

- テキストと背景のコントラスト比: 最低4.5:1（AA準拠）
- 重要な情報は色だけでなく形状やアイコンでも区別

### タッチターゲット

- 最小サイズ: 44x44px
- 要素間の最小間隔: 8px

### 目への優しさ

- 純粋な白（#FFFFFF）は使わない → `--color-text-primary: #f0eef5`
- 高コントラストを避け、夜間の使用に配慮

---

## 8. 実装時の注意点

### やるべきこと

- デザイントークン（CSS変数）を必ず使用し、ハードコードを避ける
- 背景には微細なグラデーションやパーティクル効果を入れる
- 角丸は大きめに取り、柔らかい印象を維持する
- アニメーションはゆっくりめに設定する

### 避けるべきこと

- 純粋な黒（#000000）や白（#FFFFFF）の使用
- 角張ったデザイン、シャープなエッジ
- 派手な色、高彩度のアクセント
- 速すぎるアニメーション、急な画面遷移
- テック感のあるフォント（Inter, Roboto, SF Pro等）
