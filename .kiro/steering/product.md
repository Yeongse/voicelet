# Product Overview

Voiceletは音声投稿に特化したソーシャルアプリケーション。ユーザーが短い音声クリップを録音・共有できるプラットフォームを提供する。

## Core Capabilities

- **音声録音・投稿**: 30秒以内の音声を録音し、他のユーザーと共有
- **認証なし体験**: デモユーザーとして即座に機能を試用可能
- **マルチプラットフォーム**: モバイル（iOS/Android）向けネイティブアプリ

## Target Use Cases

- 短い音声メッセージの共有
- 音声コンテンツの作成・消費
- 認証前のアプリ体験（デモモード）

## Value Proposition

テキストではなく音声を主体としたコミュニケーションを実現。登録前に機能を体験できるデモモードにより、ユーザー獲得の障壁を下げる。
